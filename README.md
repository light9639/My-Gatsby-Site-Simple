## 🚀 Quick start

1.  **Gatsby 사이트 생성.**

    Gatsby CLI를 사용하여 새 사이트를 생성합니다.

    ```shell
    # create a new Gatsby site using the minimal starter
    npm init gatsby
    ```

2.  **개발 시작.**

    새 사이트의 디렉토리로 이동하여 시작하십시오.

    ```shell
    cd my-gatsby-site/
    npm run develop
    ```

3.  **파일을 열고 편집하십시오!**

    사이트는 현재 http://localhost:8000/ 에서 실행 중입니다!

    `src/pages/index.js`를 편집하면 실시간으로 사이트 업데이트 할 수 있습니다!
